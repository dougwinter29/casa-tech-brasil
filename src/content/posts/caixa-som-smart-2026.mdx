## Como usar a caixa de som para controlar a casa

**Comandos básicos:**
```
"Alexa, apaga todas as luzes"
"Alexa, coloca o ar a 23 graus"
"Alexa, liga a câmera da sala" (Echo Show)
"Alexa, toca jazz no Spotify"
"Alexa, define alarme para 7h"
"Alexa, qual a previsão do tempo?"
```

**Rotinas de voz:**
```
"Alexa, bom dia" → liga luzes + fala temperatura + toca música
"Alexa, boa noite" → apaga luzes + desliga TV + ativa segurança
"Alexa, modo cinema" → reduz luzes + liga TV + coloca 23°C
"Alexa, cheguei em casa" → liga ar + acende sala + toca playlist
```

## Multi-room: música em todos os cômodos

Com múltiplos dispositivos Echo, você cria um grupo e toca **a mesma música sincronizada em toda a casa**:

1. No app Alexa → Devices → Multiroom Music
2. Crie um grupo com os Echos que quiser
3. "Alexa, toca música em casa toda" → todos tocam sincronizado

É uma das experiências mais impressionantes que a casa inteligente proporciona.
