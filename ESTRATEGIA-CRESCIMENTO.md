## 💰 Estratégia de Monetização

### Fase 1: Mercado Livre Afiliados (Imediato)

1. Acesse: mercadolivre.com.br/afiliados
2. Crie sua conta de afiliado
3. Substitua os placeholders `#afiliado-*` nos artigos por links reais
4. Foque nos produtos com maior ticket: robôs aspiradores (R$1.000+), fechaduras (R$500+)

**Meta:** R$500/mês nos primeiros 3 meses

### Fase 2: Amazon Afiliados (Mês 2)

1. Acesse: associados.amazon.com.br
2. Adicione links Amazon nas páginas de produtos americanos (Alexa, iRobot, Arlo)
3. Comissão: 3-8% por categoria

**Meta:** R$300/mês adicionais

### Fase 3: Google AdSense (Mês 4+)

Requer: Mínimo 50-100 artigos, tráfego consistente, política cumprida.

1. Configure `NEXT_PUBLIC_ADSENSE_ID` no Vercel
2. Adicione o componente `<AdSense />` nas páginas de artigos

**Meta:** R$1.000/mês após aprovação

### Fase 4: Produtos Digitais (Mês 6+)

- E-book: "Guia Definitivo de Casa Inteligente Brasileira" (R$37-97)
- Curso online: "Monte sua Casa Inteligente do Zero" (R$197-497)

## 📱 Estratégia de Redes Sociais

### Pinterest (ALTA PRIORIDADE)

Pinterest é o canal com melhor ROI para nichos de casa e decoração.

**Ações:**
1. Crie conta empresarial: pinterest.com/business
2. Crie Boards temáticos:
   - "Casa Inteligente Brasil"
   - "Robôs Aspiradores"
   - "Iluminação Smart"
   - "Segurança Residencial"
   - "Automação Barata"
3. Crie Pins para cada artigo com imagem vertical (1000x1500px)
4. Texto SEO no pin: título + palavras-chave
5. Link direto para o artigo

**Template de Pin:**
- Fundo tecnológico (azul escuro ou preto)
- Título chamativo em destaque
- Imagem do produto principal
- Logo Casa Tech Brasil
- URL do site

**Frequência:** 5-10 pins por dia (pode repetir pins antigos)
**Ferramenta gratuita:** Canva + Pinterest Scheduler

### YouTube Shorts / TikTok / Reels

**Formato:** 30-60 segundos, vertical (9:16)

**Roteiro padrão para Shorts:**
```
0-5s: Hook ("Você sabia que pode montar uma casa inteligente por R$500?")
5-25s: 3 produtos rápidos com preço e link
25-30s: CTA ("Link na bio para o guia completo")
```

**Temas de Shorts:**
1. "3 gadgets que todo brasileiro deveria ter" (custo < R$200)
2. "Esse robô aspirador mudou minha vida" (review rápido)
3. "Como a Alexa pode salvar seu dia" (demonstração real)
4. "5 automações que você pode fazer HOJE" (tutorial rápido)
5. "Produtos inteligentes que você não sabia que precisava" (descoberta)

**Frequência:** 1 vídeo por dia nos primeiros 3 meses

## 🚀 Deploy Vercel — Passo a Passo

1. **Criar conta:** vercel.com/signup (gratuito)
2. **Login via CLI:**
   ```bash
   cd casa-tech-brasil
   npx vercel login
   ```
3. **Deploy:**
   ```bash
   npx vercel deploy --prod
   ```
4. **Configurar domínio:**
   - No painel Vercel: Settings → Domains
   - Adicionar `casatechbrasil.com.br`
   - Configurar DNS no registro do domínio:
     - Tipo A: 76.76.19.61 (Vercel IP)
     - Ou CNAME: cname.vercel-dns.com
5. **Variáveis de ambiente:** Settings → Environment Variables
   - Adicionar as variáveis do `.env.example`

### Hospedagem (plano gratuito Vercel inclui):
- SSL/HTTPS automático
- CDN global
- Deploy automático no git push
- 100GB de bandwidth/mês
- Custom domains ilimitados
